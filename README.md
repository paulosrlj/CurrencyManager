# Expenses Manager
> Um app para gerenciamento de despesas.<br>

[![Build Status][travis-image]][travis-url]

Os dados são persistidos no localStorage do navegador<br>

![](preview.png)

## Instalação

### Instalar dependências

```sh
npm i
```

### Rodar

```sh
npm start
```
